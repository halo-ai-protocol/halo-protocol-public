## What Is HALO?

Every AI conversation you have today starts at zero.

Every session is amnesic. Every model — regardless of capability — has no idea who you are, what you've built, what you've decided, or what belongs to you intellectually.

**HALO is the protocol layer that ends that.**

HALO gives every human a persistent, attributable, and growing knowledge identity across all AI engagements — regardless of model, platform, or application. You own it. You control it. You take it with you.

---

## The Protocol Stack

| Sub-Protocol | Purpose |
|---|---|
| **HALO/PACP** | Personal AI Context Protocol — your portable knowledge identity |
| **HALO/UACP** | Universal AI Context Protocol — organizational knowledge layer |
| **HALO/CRAG** | Crystallized RAG — knowledge pattern crystallization engine |
| **HALO/TDO** | Transient Data Object — transient knowledge events made permanent |
| **HALO/TDOM** | Transient Data Object Model — cross-session knowledge map |
| **HALO/AR** | Attribute Registry — canonical artifact storage, attributes not containers |
| **HALO/LSTO** | Local Storage Task Object — session-portable task context |

## Why This Matters

The internet has transport (HTTP). It has addressing (DNS). It has identity (barely). It has rendering (HTML, CSS, JS).

What it has never had is a **persistent, attributable human knowledge layer for AI.**

HALO is that layer.

Not for one AI. Not for one platform. For all of them.

---

## Core Principles

- **Attributes, not containers** — the 50-year-old folder model ends here
- **Attribution is immutable** — provenance is baked in at ingestion, permanently
- **Knowledge crystallizes** — patterns that prove themselves become fast, named, versioned queries
- **Owner sovereignty** — your knowledge identity is yours, not the platform's
- **Platform agnostic** — one spec, any compliant infrastructure
